# Jet Game

Navigate through the obstacles while the timer runs down. Go until you lose.

Download the [zip](https://github.com/bcout/unity/blob/main/Jet%20Game/Jet%20Game.zip) to play!

# Controls

Arrow keys to move.
P, A, D, S to toggle lighting
